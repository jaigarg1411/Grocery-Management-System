Instructions to run the program:-

1. Open terminal.
2. Go to the directory in which all program files are kept.
3. Type make on terminal.
4. Wait for a while, until object files and executable is made.
5. Type './grocery' on terminal.
6. Various functionalities will be appeared on the screen.
7. Select the functionality you want to execute.
8. Follow the instructions.
9. Type '0' to EXIT the program.
